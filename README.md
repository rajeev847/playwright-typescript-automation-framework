# Playwright TypeScript Automation Framework

## Overview

This project demonstrates a scalable UI Automation Framework built using Playwright and TypeScript following Page Object Model (POM) design principles.

The framework is designed to automate end-to-end business workflows, validate application functionality, and support cross-browser testing across Chromium, Firefox, and WebKit browsers.
